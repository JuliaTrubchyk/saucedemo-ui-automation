# SauceDemo UI Automation (Selenium + TestNG)

UI test automation project for https://www.saucedemo.com/

## Tech Stack
- JDK: 21 Oracle OpenJDK 21.0.3
- Selenium 4.39.0
- TestNG 7.11.0
- Maven
- IntelliJ IDEA

## How to run (will be updated)
- Run tests: `mvn test`

## Project goal
Build a clean, maintainable UI automation framework (Page Object Model, stable waits, readable tests).
